❓ Dúvidas

Perguntas frequentes.

A ferramenta é realmente gratuita? +
Sim. O download e o uso da plataforma são 100% gratuitos. Você paga apenas pelos tokens dos modelos de IA que escolher usar — como Claude, GPT ou Meta. Não há mensalidade, não há assinatura, não há cobrança escondida pela ferramenta em si.
Minhas credenciais SAP ficam seguras? +
Absolutamente. O app é desktop e se conecta diretamente ao seu SAP via ADT (ABAP Development Tools). Suas credenciais ficam armazenadas apenas localmente, criptografadas no seu computador. Elas nunca são enviadas para nenhum servidor externo.
Precisa de acesso especial no SAP? +
A ferramenta utiliza a API ADT (ABAP Development Tools) do SAP, que já está disponível em sistemas SAP modernos (NetWeaver 7.4+). Você precisa de um usuário com acesso de leitura ao repositório ABAP. Nenhuma instalação de add-on é necessária no SAP.
Quais modelos de IA estão disponíveis? +
A plataforma suporta mais de 10 modelos: Claude Sonnet 4.6 (Anthropic), GPT-5.4 e GPT-4.1 (OpenAI), Meta Maverick 4 (ultra econômico), entre outros. Cada modelo exibe o custo por token, e você escolhe qual usar em cada conversa conforme a complexidade e o custo desejado.
Funciona com SAP on-premise e BTP? +
Sim. A ferramenta suporta conexões com SAP on-premise (via IP/host) e SAP BTP (Business Technology Platform). Você pode configurar múltiplos ambientes (DEV, QAS, PRD) e alternar entre eles sem sair da aplicação.
A IA pode modificar ou deletar dados no SAP? +
Não. A ferramenta opera apenas em modo leitura. O SQL Executor permite apenas queries SELECT — comandos INSERT, UPDATE e DELETE são bloqueados. A IA acessa dados para análise, mas não tem capacidade de alterar seu sistema SAP.
Funciona em Windows e macOS? +
Sim. O app é construído com Tauri v2, o que garante compatibilidade nativa com Windows e macOS. É um executável nativo — não é um webapp embutido — o que significa performance e integração real com o sistema operacional.
