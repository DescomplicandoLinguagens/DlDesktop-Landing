✦ Segurança & Privacidade

Seus dados SAP
ficam com você

O DLAcademy foi projetado com privacidade como prioridade. Seus dados de produção nunca saem da sua máquina.

✓
Credenciais armazenadas localmente
IP, usuário e senha do SAP ficam criptografados apenas no seu computador. Nunca são enviados para servidores externos.
✓
Consultas diretas ao seu SAP
As chamadas ADT vão diretamente do seu app para o seu SAP. Nenhum dado de produção passa pelos servidores da DL.
✓
Apenas validação de licença online
O único dado que trafega externamente é a validação da sua chave de API para verificar a licença ativa.
✓
Sessão persistente local
Sua sessão é salva localmente via armazenamento seguro. Sem login repetido, sem dados na nuvem.

✦ Perguntas frequentes

Tirou as dúvidas?

O que é a chave de API e como eu obtenho?
A chave de API é fornecida pela Descomplicando Linguagens após a contratação da licença. Ela é usada apenas para validar que você tem acesso ativo à ferramenta. Entre em contato com o suporte da DL para solicitar a sua.
Meus dados do SAP ficam nos servidores da DL?
Não. As credenciais SAP são armazenadas localmente no seu computador e as consultas são feitas diretamente do app para o seu SAP. O único dado que trafega externamente é a validação da sua chave de API para verificar a licença.
O app funciona com SAP on-premise e BTP Cloud?
Sim. O DLAcademy suporta ambientes on-premise via ADT (basta informar IP, mandante e credenciais) e SAP BTP Cloud ABAP (via Service Key JSON). Você pode cadastrar múltiplos ambientes e alternar entre eles livremente.
Posso usar o chat sem estar conectado a um SAP?
Sim, mas as perguntas que exigem consulta ao SAP não poderão ser respondidas sem um ambiente ativo. O chat pode responder perguntas gerais sobre ABAP e SAP mesmo sem conexão.
O Windows mostra aviso de segurança ao instalar. É normal?
Sim, é normal em aplicativos novos sem certificado EV. Clique em "Mais informações" e depois em "Executar assim mesmo". No macOS, vá em Preferências do Sistema → Privacidade e Segurança → clique em "Abrir assim mesmo".
O app se atualiza automaticamente?
As atualizações são disponibilizadas como novos instaladores. Você será notificado dentro do app quando houver uma nova versão disponível.
